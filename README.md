
## idea-bookmark

本插件可以将idea的书签同步到特定的文件里面， 默认为 `${projectDir}/.idea/bookmarks.json` 文件下面， 并且会随着书签的变更自动同步

> 建议将此文件从 `.gitignore` 文件中忽略, 避免书签丢失
